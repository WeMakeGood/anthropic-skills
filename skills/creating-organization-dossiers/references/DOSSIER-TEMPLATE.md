## Section Guidance

### When to Include Each Section

| Section | Always | If Available | If Requester Context |
|---------|--------|--------------|---------------------|
| Executive Summary | Yes | | |
| Organizational Overview | Yes | | |
| Programs & Services | Yes | | |
| Leadership & Governance | | Board details | |
| Financial Profile | | 990 data | |
| Partnerships | | If found | |
| Digital Presence | | Website scraped | |
| Strategic Analysis | | | Yes |
| Information Gaps | Yes | | |

### Adapting for Purpose

**Prospect Research:**
- Emphasize financial health and leadership
- Focus on strategic alignment potential
- Include detailed partnership analysis

**Grant Application Research:**
- Focus on programs and outcomes
- Highlight mission alignment
- Note funder relationships

**Partnership Evaluation:**
- Emphasize complementary capabilities
- Analyze strategic fit
- Identify collaboration opportunities

**General Background:**
- Balanced coverage of all sections
- Less emphasis on strategic analysis
- More emphasis on factual overview
